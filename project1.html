The cycle of life

Background

When I read the description of the home work "Come up with a natural system you'd like to represent in code‚Äù, I thought about what is the most natural phenomenon in this world. In my opinion, it is the cycle of life, it is the birth, growth, maturity to old age. No natural creature escapes such a law, nor man, nor trees. I have read about recursive coding before, and I would like to take this opportunity to try to apply this code, so I finally decided to represent the process of trees growing in all seasons in code.

Diagram(original version)

At first, I wanted to focus on autumn because I can code the recursive tree and has some interactive leaves. At this stage, I didn't have a lot of ideas about interaction, but simply wanted to control the direction with the mouse. So I just made the following diagram.

Diagram(revised version)

But after the first week of critique, I gained a lot of interesting interaction ideas. I plan to change the season by clicking the mouse and control the speed and direction of the wind with the position of the mouse. So I got a second version of the diagram.

Coding process

In the process, making the diagram helped me a lot to deal with the complexity and have a deeper understanding of the structure of the code. In the diagram, I listed the elements that should appear in the picture: the background, the tree, and the leaves. In this way, I split the code into three sections: the main section, the leave section, and the tree section.

The other way to deal with the complexity is to simplify the code at first to focus on the function. In the beginning, I made my code in black and white colors and graphics and after I figured out how to make it functional, I add the visual on that base.

By studying the teacher's demos and online examples, I finally achieve the initial interaction, and here is my final code.
